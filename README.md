# FameFi Creator Economy Platform

A decentralized platform for creators to tokenize and monetize their digital content on the blockchain.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
- React Router
- Framer Motion

## Development Setup

To run this project locally:

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

## Project Overview

FameFi is a creator-first SocialFi protocol built on Solana that empowers creators with on-chain ownership and fair monetization. Turn your viral content into tradeable on-chain assets and join a thriving ecosystem of creators, collectors, communities, brands, and investors.
